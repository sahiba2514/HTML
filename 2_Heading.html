Unicode is a universal character set that defines all the characters needed for writing the majority of living languages in use on computers. Unicode aims to be (and already is) a superset of all other encoded computer character sets. The Unicode Standard covers (almost) all characters, punctuations, and symbols in the world and enables processing, storage, and transport of text independent of platform and language.
